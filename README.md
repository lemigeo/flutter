## install

## run in debug mode
create .env file and set AUTH_URL and API_URL.   
flutter pub get.  
set flutter device.  
press F5.  

## related M1 chip
set platform version in the ios/Podfile: platform :ios, '10.0'

sudo arch -x86_64 gem install ffi
arch -x86_64 pod install
